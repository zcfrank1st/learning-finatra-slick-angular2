Finatra

Finatra is a Scala service framework built on top of twitter-sever and Finagle.

This template shows the scaffolding for a simple Http server with a single controller. See the Finatra Http overview for more information and examples.

Full documentation available on the Finatra project site.

Configuring your application

The build.sbt contains the sbt build configuration; you'll want to edit at least the project name and version.

This example by default will run your external service on port :9999 -- you can override this by either changing the code in the ExampleServer class or by passing a different value in the -http.port flag.

Other Examples

More examples are available in the Finatra Examples project on github.
